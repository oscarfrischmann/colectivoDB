Precios USD para cursos de ingles

  • Ingresar 1 o varios precios.
  • Si se deja el valor vacío, se mantiene el valor anterior.
  • Si se ingresa el número 0 (cero) se mostrará el valor del curso como "---" en pantalla
  • El criterio es ingresar el valor más alto (clases por hora, no pack; curso x mes; etc) y esperar que consulten por las "promociones" (pack 8 horas, pago curso completo, etc)

Pop Up

Configuración de Pop Up

  • Para cambiar la imagen, subir una nueva, ver imagenes, usar.
  • Se puede usar sólo imagen, para eso, escribir el número 0 en los otros campos
  • Los campos en 0 NO SE MUESTRAN, por ejemplo, si ahora vas a agregar todo un popup nuevo, completas todos los campos y estos van a reemplazar
  • Si no querés el botón de OK (confirm, lo usamos para redirigir, por ejemplo a CURSOS) escribir el numero 0(cero) --- SIEMPRE hay que ingresar 0 o el texto del botón
  • La url de redirección es la acción del botón de OK, por eso lo anulamos con 0 si queres que una vez visto el popup se siga en la home
  • Para cambiar la URL de redirección, seleccionar una de las opciones del menú desplegable
  • Para actualizar 1 solo campo, redactar ese campo ( o imagen ) y dejar los otros campos en blanco
  • El botón "Mostrar popUp?" si querés que haya un popup va en SI, sino queda en NO
  • cualquier duda consultame!!!
